<h1>🎬 Clone da Netflix (Projeto Front-End)</h1>
<p>Este projeto é uma recriação da interface da Netflix, desenvolvido com foco em praticar conceitos de HTML e CSS, além de simular a navegação entre páginas de uma aplicação real.</p>

<h2> 🚀 Funcionalidades </h2>

  <p> 🔐 Tela de Login</p>
  <p>👤 Seleção de Perfis</p>
  <p>🎥 Página Principal com destaques e listas de filmes</p>
  <p>🎨 Interface inspirada no design da Netflix</p>
  <p>🖱️ Efeitos visuais com hover e active</p>

<h2>🎯 Objetivo</h2>

<p>Este projeto foi desenvolvido com o objetivo de:</p>
<p>- Praticar estruturação de páginas com HTML</p>
<p>- Trabalhar estilização com CSS</p>
<p>- Simular uma aplicação real de streaming</p>
<p>- Desenvolver habilidades de organização de código</p>
